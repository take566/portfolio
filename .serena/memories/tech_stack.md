# 技術スタック

## プログラミング言語
- **Python**: メイン言語

## 主要依存関係
- **pandas (>=1.5.0)**: データ処理・Excel読み込み
- **openpyxl (>=3.0.0)**: Excel形式ファイルの読み書き
- **tabulate (>=0.8.0)**: テーブル形式出力

## ツール・環境
- **Git**: バージョン管理
- **PowerShell**: Windowsでのコマンド実行
- **pip**: パッケージ管理
- **仮想環境**: `.venv`ディレクトリで管理

## ファイル形式
- **入力**: Excel (.xlsx, .xls)
- **出力**: Markdown (.md)
- **エンコーディング**: UTF-8
