## 主要技術スキル

### クラウド・インフラ

- **AWS**: ECS, ECR, Lambda, RDS, DynamoDB, CloudFormation, SAM, CodeCommit, CodePipeline
- **Azure**: Virtual Desktop, Power Platform, Azure Repos, Azure Pipeline, Web App Service
- **Kubernetes**: 基盤構築、運用
- **Docker**: コンテナ化、オーケストレーション

### 仮想化・VDI

- **VMware**: vSphere 5.0-6.7, Horizon View
- **Citrix**: XenDesktop 5.6-7.6, XenApp 6.0-7.15
- **Hyper-V**: Windows Server環境
- **KVM**: Linux仮想化

### DevOps・CI/CD

- **Jenkins**: パイプライン構築
- **GitLab**: CI/CD、リポジトリ管理
- **GitHub Actions**: ワークフロー自動化
- **Ansible**: 自動化、設定管理
- **Terraform**: Infrastructure as Code

### データベース

- **MySQL**: 運用・構築
- **PostgreSQL**: 運用・構築
- **SQL Server**: 2008-2017
- **Oracle**: 基盤システム
- **DynamoDB**: NoSQL

### プログラミング言語

- **Python**: 3.6-3.9
- **Java**: 8
- **Go**: サーバレス開発
- **PHP**: Web開発
- **React**: フロントエンド

### 監視・運用

- **Zabbix**: システム監視
- **Prometheus**: メトリクス収集
- **Hinemos**: 統合監視
- **CloudWatch**: AWS監視

## プロジェクト管理経験

- **PL/PM**: 最大100名規模のプロジェクト管理
- **要件定義**: 顧客との折衝、要件整理
- **設計**: 基本設計、詳細設計、運用設計
- **構築**: 環境構築、テスト実施
- **運用**: 保守・改善、監視体制構築
